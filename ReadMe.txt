Morse Code
Morse code is an application which encodes English phrases into Morse Code

Installation
At the moment this application is in its initial phases so you just need to build and run the code

Usage
Just run the application and input any English phrase that you prefer to encode into Morse Code
Contributing
This mini-project is done by a group of students . If you think that something can be improved we're open to discuss it with you.
